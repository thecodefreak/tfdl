# Tool Template

Copy this folder into the target category and rename it to your tool slug.

Checklist:

1. Update page title and description.
2. Update the `<wt-tool-shell>` attributes (`eyebrow`, `heading`, `lede`) in `index.html`.
3. Register the tool in `assets/js/tools.registry.js`.
4. Set the alias in the registry entry (`alias` field).
5. Keep canonical path stable: `tools/<category>/<slug>/`.
